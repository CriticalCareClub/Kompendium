Hier zählt der Eindruck, den der Patient auf den Helfer macht. Es werden
v. a. beurteilt:

-  Alter, Geschlecht

-  Hautkolorit (Hautfärbung)

-  Haltung, Gesichtsausdruck, spontane Bewegungen

-  Sprache

-  Offensichtliche Verletzungen, Blutungen

-  Sonstige Aufffälligkeiten

Liegt der Verdacht auf eine Wirbelsäulenverletzung vor, wird bereits
hier mit der manuellen Stabilisierung der Halswirbelsäule begonnen
(:ref:`Thema-IndikationWirbelsaeulenimmobilisation`).




Befunde
=======

Auffallen kann eine *blasse Haut*, *Zyanose*, *Schweiß*, *Schonhaltung*,
*Atemnot* (Gestik, Aufstützen der Arme um die *Atemhilfsmuskulatur*
einzusetzen, Nasenflügeln, Einziehungen an
den Rippen, …), eine verwaschene oder wirre *Sprache*,
*Desorientiertheit*, *Krämpfe* u.v.a.m. Ebenso können eventuell Hinweise
auf ein *Trauma*, sichtbare Verletzungen, starke Blutungen oder Hinweise
auf eine Wirbelsäulenverletzung erkannt werden.
