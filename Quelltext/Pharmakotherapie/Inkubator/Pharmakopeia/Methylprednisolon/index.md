# Methyl-Prednisolon

```{index} single: Prednisolon
```

```{eval-rst}

:Indikation(en):

:Kontraindikation(en):

   Rückenmarkstrauma

:Dosierung: Bolus: 30 mg / kg in 15 min, Erhaltung: 5,4 mg / kg / h über 23 h ( / 47 h) (NASCIS)

:Kontraindikation(en):

   - I. Notfall SHT,

   - sonst Überempfindlichkeit,

   - Pilz-, Herpes-, TB-Infektion,

   - ak. Psychosen,

   - Ulcera

:Anmerkungen:

   Therapiebeginn innerhalb der ersten 8 h nach Querschnittslähmung Wirkung beim WS-Trauma umstritten (ÖNK Consensus 2005 3 : präklinisch nicht empf.),
   wenn, dann nur Methyl-Prednisolon! NASCIS-Schema: Bolus (30 mg / kg) nach < 3 h, spätestens < 8 h, gefolgt von Erhaltungs-Infusion 5,4 mg / kg / h für 23 h (bzw. 47 h, Rx erst ≥ 3 h nach Trauma).
```
