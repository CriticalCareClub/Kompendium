# Ⓒ Hämodynamisches Monitoring und Diagnostik



Die Überwachung ist einer der grundlegenden Aspekte der Intensivtherapie.
Sie dient dazu

- um uns über eine Verschlechterung oder mögliche Verschlechterung des Zustands des Patienten zu informieren
- einzugreifen, wenn es Anzeichen für eine Verschlechterung gibt
- um das Ansprechen auf die Behandlung zu beurteilen

Dabei hat das hämodynamische Monitoring auch die wesentliche Funktion einer hämodynamischen **Diagnostik**, welche uns Anhaltspunkte für eine zielgerichtete Therapie geben kann.
