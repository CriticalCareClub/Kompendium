% Im Text muss bei der Fußnote, auf die ein zweites mal verwiesen werden soll, ein Label (hier: \label{fn:wichtig})eingefügt werden.
% Mit \footref{fn:wichtig} verweist man dann ein weiteres mal auf sie.
